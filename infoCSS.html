HTML создает структуру, а CSS позволяет стилизовать

    Как сязать HTML & CSS

  1. создать отдельный файл с расширением .сss и затем в html-файле в головной части указать link rel = "stylesheet" href = "style.css"
  2. использовать закрывающийся тег style в голове: 
    li{
                font-family: Georgia;
                font-size: 18px;
                color: gray;
            }
  3. использовать в описании раздела style: 
    4.  p style="font-style:  italic "

Чтобы перестраховаться от таких ситуаций, когда при разработке сайта использовали такой шрифт, который не установлен на компьютере, необходимо использовать такое понятие как fonts fallback "фоллбэк". 

 p{font-family:"Times New Roman", Georgia, serif;}
К тексту параграфа будет применен шрифт Times New Roman Если же вдруг у пользователя нет такого шрифта на компьютере, тогда все тексты параграфов необходимо отображать шрифтом Georgia.
Если же и этот шрифт не найдется на компьютере пользователя, тогда отображать любым стандартным шрифтом, относящимся к группе шрифтов с засечками (serif).

чтобы поменять регистр текста используется text-transform.

    ТИПЫ text-tranform:

  • text-tranform:capitalize (Первая Буква Каждого Слова Будет Большой)
  • text-tranform: lowercase (все буквы будут маленькими)
  • text-transform: uppercase (ВСЕ БУКВЫ БУДУТ БОЛЬШИМИ)
  • text-tranform: inherit (наследует форматирование родителя)
  • text-transform: none (отменяет предыдущие форматирование)

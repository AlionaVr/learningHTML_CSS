Unordered List

    Варианты маркеров :

  • list-style-type:circle - маркер будет отображаться кружочком
  • list-style-type:square - будет выглядеть как квадратик
  • list-style-type:none - список будет без маркеров.

Ordered List

    Варианты нумерации:

  1. type="A" - строки будут нумероваться заглавными буквами
  2. type="a" - строчными буквами
  3. type="I" - заглавными римскими цифрами
  4. type="i"- строчными римскими цифрами
